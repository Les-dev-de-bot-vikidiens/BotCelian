# BotCélian
### Description

> [!NOTE]
> BotCélian est un bot dévellopé pour les wikis Mediawiki avec Piwikibot, il fonctionne uniquement sur Vikidia en français pour le moment.

### Scripts
Les scripts de BotCélian :

* __```ArticleMAJ.py```__ : Permet de mettre à jour la lister des _Articles importants et courts_ en focntion de leurs tailles en octets.
* __```Stats.py```__ : Permet de mettre à jour les statistiques du jour (sur [cette page](https://fr.vikidia.org/wiki/Utilisateur:BotCélian/Stats))
* __```categinex.py```__ : Permet de supprimer les catégories inexistantes des pages encyclopédiques (script de [Muffy](https://fr.vikidia.org/wiki/Utilisateur:BotCélian/Stats](https://github.com/orgs/Les-dev-de-bot-vikidiens/people/luffythebest-37)))
* __```ebauche_24.py```__ :  Permet d'ajouter des ébauches en fonctions des portails sur les pages modifiées dans les dernières 24 heures.
* __```ebauche_auto.py```__ : Permet d'ajouter des ébauches en fonctions des portails sur les pages crées dans les dernières 24 heures.
* __```ebauche_teste.py```__ :  Permet d'ajouter des ébauches en fonctions des portails sur des pages aléatoires.
* __```ebauche_titre.py```__ : Permet d'ajouter des ébauches en fonctions des portails sur des pages choisis par l'utilisateur (surtout utiliser pour tester le script)
* __```homonyme.py```__ : Permet de supprimer les portails des pages d'homonymies de Vikidia (script de [Muffy](https://fr.vikidia.org/wiki/Utilisateur:BotCélian/Stats](https://github.com/orgs/Les-dev-de-bot-vikidiens/people/luffythebest-37)))
* __```mass_rollback.py```__ : Permet de révoquer un grand nombre de modifiactions d'un utilisateur sur une période précise (```Ne fonctionne pas bien```)
* __```personnality.py```__ : Permet d'ajouter ```Catégorie:Personnalité masculine par ordre alphabétique``` ou ```Catégorie:Personnalité féminine par ordre alphabétique``` des articles de la catégorie ```Catégorie:Personnalité par ordre alphabétique``` en fonction d'une requête SPRQL à WikiData
* __```personnalityDoublon.py```__ : Permet de supprimer la catégorie ```Catégorie:Personnalité par ordre alphabétique``` si ```Catégorie:Personnalité masculine par ordre alphabétique``` ou ```Catégorie:Personnalité féminine par ordre alphabétique``` est présente sur la page.
* __```typo.py```__ : permet de corriger la typo et d'autres petites choses sur des articles (en cours de dévellopement)
